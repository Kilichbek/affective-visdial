Phase: Dev Phase

Starts on: Aug 25, 2023

Ends on: Sep 25, 2023

During the Dev Phase, which lasts approximately 3 weeks, each team uses the provided training set and public test set to build and test their models.

The leaderboard will reflect the scores obtained on the public test set.

Max submissions/day: 5

Max submissions/month: 50

Max total submissions: 50

Max concurrent submissions: 3

Phase: Test Phase

Starts on: Sep 10, 2023 3:00:00 AM +03 (GMT + 3:00)

Ends on: Sep 16, 2023 2:59:59 AM +03 (GMT + 3:00)

During the Test Phase, which takes place in the final 1 week before the submission deadline, we will release a hidden test set of a similar size to the public test set. Teams can then submit their results for this hidden test set.

The final winners will be decided based on this set.
