Evaluation Criteria

Submissions will be evaluated based on the following criteria:

  • Accuracy of predicted emotional labels.
  • Quality of generated explanations.

Automatic Evaluation on Test Set

We will use the following metrics for automatic evaluation:

  • Weighted F1
  • BLEU
  • BERT score

Evaluation scores will be computed for each submission using these metrics.

Evaluation Platform: Eval.AI

We will use the Eval.AI platform to perform automatic evaluation and display the results via a leaderboard.

The leaderboard scores will be sorted based on the Weighted F1 score, providing a comprehensive ranking of submissions.
